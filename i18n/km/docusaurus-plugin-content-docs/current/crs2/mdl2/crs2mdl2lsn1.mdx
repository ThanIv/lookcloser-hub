### ជំហានទី 1: ការចូលប្រាស់

- **Desktop (PC/Laptop)**:
    - ចូលទៅកាន់ [images.google.com](https://images.google.com/)
    - ចុចរូបតំណាងកាមេរ៉ានៅក្នុងរបារស្វែងរក
    - ជ្រើសរើស Upload an image → ជ្រើសយក File → ចុចលើ image
- **Mobile (Phone/Tablet):**
    - បើកកម្មវិធី Chrome
    - ចុចរូបតំណាងកាមេរ៉ា
    - ជ្រើសរើស Upload an image → ជ្រើសយក photo ពីក្នុង gallery ឬកន្លែងផ្ទុក

### Step 2: មើលលទ្ធផលដែលបង្ហាញ

នៅពេល Google បញ្ចប់ការស្វែងរក, អ្នកនឹងឃើញ:
- **រូបភាពស្រដៀងគ្នា** - រូបភាពផ្សេងទៀតដែលមើលទៅស្រដៀងឬដូចរូបភាពរបស់អ្នក។
- **គេហទំព័រដែលទាក់ទង** - ដែលជាកន្លែងដែលរូបភាពត្រូវបានប្រើប្រាស់នៅលើអ៊ីនធឺណិត
- **បង្ហាញពេលវេលា** - ពេលខ្លះបង្ហាញនៅពេលដែលវាត្រូវបានប្រើលើកដំបូង

ការស្វែងរកប្រភពនៃរូបភាពនឹងមិនដោះស្រាយបញ្ហាបានទាំងអស់នោះទេ ប៉ុន្តែវាផ្តល់ឱ្យយើងនូវវិធីដ៏មានប្រសិទ្ធិភាពក្នុងការពិនិត្យមើលការពិតនូវអ្វីដែលយើងឃើញនៅលើអ៊ីនធឺណិត។ គន្លឹះសំខាន់គឺត្រូវគិតដល់ការស្វែងរកប្រភពដើម គិតឱ្យបានល្អិតល្អន់ និងពិនិត្យមើលឡើងវិញមុនពេលចែករំលែក។

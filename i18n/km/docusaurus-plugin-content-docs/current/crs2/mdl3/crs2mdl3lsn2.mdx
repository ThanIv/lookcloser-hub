Click on Tools under the Google Images bar, then select Time. This lets you filter results by date. If you want to know when an image first appeared, look for the earliest date. This is especially useful to detect recycled old photos used as ‘breaking news’.
- Click Tools → Time to filter results
- See when the image first appeared
- Spot reused or recycled photos
